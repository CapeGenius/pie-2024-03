Mechanical System

Wheel Torque

Measured wheel torque to be at least 7 N-M by clamping scale to edge of wheel and running wheel at max PWM/40V. Clamp does not stay on tire at more than 7 N-M, so need to test by clamping on to the end of the rail and not to the tire to read max torque. Based on 25mm stators with 4 wraps, should get 12.7 N-M if drawing 15 amps.

All our code for this project is in this github repository.

PWM Motor Speed Control

The code controls balancing system using a PID controller and accelerometer/gyroscope. It reads tilt angle from gyroscope and adjust motor speed and direction based on the setpoint and current tilt using PID. In the main loop, it continuously computes the tilt angle, adjusts motor speed and direction based on the PID output, and stops the motor when the brake switch is pressed. The motor direction and speed are updated in real-time.
